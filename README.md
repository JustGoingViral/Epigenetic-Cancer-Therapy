## 🛠️ Installation

### Prerequisites
- Python 3.8+
- Node.js 16+
- Docker and Docker Compose
- PostgreSQL 13+

### Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-org/Multi-Targeted-Epigenetic-Cancer-Therapy.git
   cd Multi-Targeted-Epigenetic-Cancer-Therapy
   ```

2. **Set up the backend**
   ```bash
   cd backend
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   pip install -r requirements.txt
   ```

3. **Set up the frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```

4. **Start with Docker (Recommended)**
   ```bash
   docker-compose up -d
   ```
